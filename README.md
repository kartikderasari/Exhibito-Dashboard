# Exhibito-Dashboard

Exhibito Dashboard is a Single Page Application that will enable you to manage your [Exhibito](https://github.com/kartikderasari/Exhibito) portfolio website.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve

```
